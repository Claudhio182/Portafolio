Mis Proyectos

Sistema Control Vehículos

Este proyecto consiste en el diseño y desarrollo de un sistema de software para el control de vehículos en un sistema de peaje. La solución está basada en la programación orientada a objetos con Python e incluye modelado con diagramas de clases, implementación de clases y la manipulación de datos mediante archivos CSV.

Venta Vehículos

Este proyecto consiste en el desarrollo de una tienda virtual de venta de vehículos utilizando el framework Django. La aplicación permitirá gestionar vehículos, registrarlos en una base de datos, clasificarlos y administrarlos mediante un panel de control. Además, incluirá un sistema de autenticación y permisos de usuario..

Fabricación De Productos Farmaceuticos

Este proyecto tiene como objetivo desarrollar un sistema que simule la fabricación de productos farmacéuticos mediante el uso de Django y PostgreSQL. Se enfocará en la gestión de laboratorios farmacéuticos, directores generales y productos, estableciendo restricciones y relaciones entre las entidades.
